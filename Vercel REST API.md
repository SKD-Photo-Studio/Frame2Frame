[Using the REST API](https://vercel.com/docs/rest-api#using-the-rest-api)
Interact programmatically with your Vercel account using the SDK or direct HTTP requests. You can deploy new versions of web applications, manage custom domains, retrieve information about deployments, and manage secrets and environment variables for projects.
The API supports any programming language or framework that can send HTTP requests.

## https://vercel.com/docs/rest-api#api-basics[API basics](https://vercel.com/docs/rest-api#api-basics)
[API basics](https://vercel.com/docs/rest-api#api-basics)
The API is exposed as an HTTP/1 and HTTP/2 service over SSL. All endpoints live under the URL https://api.vercel.com and follow the REST architecture.

```
https://api.vercel.com
```

## https://vercel.com/docs/rest-api#authentication[Authentication](https://vercel.com/docs/rest-api#authentication)
[Authentication](https://vercel.com/docs/rest-api#authentication)
Vercel Access Tokens are required to authenticate and use the Vercel API. Include the token in the Authorization header:

```
Authorization
```


```
Authorization: Bearer <TOKEN>
```

Create and manage Access Tokens in your [account settings](https://vercel.com/account/tokens).

## https://vercel.com/docs/rest-api#accessing-team-resources[Accessing team resources](https://vercel.com/docs/rest-api#accessing-team-resources)
[Accessing team resources](https://vercel.com/docs/rest-api#accessing-team-resources)
By default, you can access resources in your personal account. To access resources owned by a team, append the Team ID as a query string:

```
https://api.vercel.com/v6/deployments?teamId=[teamID]
```

## https://vercel.com/docs/rest-api#rate-limits[Rate limits](https://vercel.com/docs/rest-api#rate-limits)
[Rate limits](https://vercel.com/docs/rest-api#rate-limits)
The API limits the number of calls you can make over a period of time. Rate limits are specified via response headers: X-RateLimit-Limit, X-RateLimit-Remaining, and X-RateLimit-Reset. See the [limits documentation](https://vercel.com/docs/limits#rate-limits) for details.

```
X-RateLimit-Limit
```


```
X-RateLimit-Remaining
```


```
X-RateLimit-Reset
```
