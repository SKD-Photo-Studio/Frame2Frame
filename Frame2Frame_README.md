# Frame2Frame | SKD Photo Studio Management

A high-performance, studio-management ecosystem built exclusively for **SKD Photo Studio**. This suite handles event financing, client onboarding, and team assignments with a premium, glassmorphic experience.

## 🚀 Core Features
- **Project Intelligence**: Track events, clients, and artist assignments in real-time.
- **Financial Automation**: Automated calculation of event balances, payment status, and output expenses.
- **Cloud-Native Architecture**: Built on Next.js 16 and deployed to the Cloudflare Edge for global speed.
- **Secure Multi-Tenancy**: Hardened isolation logic ensuring data integrity.

## 🛠 Tech & IT
- **Framework**: Next.js 16 + React 19
- **Database**: Supabase
- **Hosting**: Cloudflare Pages
- **Storage**: Cloudflare R2
