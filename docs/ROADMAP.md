# Future Roadmap

These features and improvements are planned for future versions of Frame2Frame but have not yet been implemented even locally.

---

## 🚀 High Priority
- **Multi-user Authentication**: Full login/signup flow for different staff members with role-based access (Editor vs. Manager).
- **Session-based Tenant Isolation**: Strictly tie API data access to the active user's session token to ensure multi-tenant security.
- **Audit Logs**: Track who made which change and when for financial accountability.

## 📊 Medium Priority
- **Advanced Export (Excel/PDF)**: Export full studio reports or event-specific invoices/vouchers.
- **Email Notifications**: Automatic alerts to clients for payment reminders or to artists for assignment dates.
- **Bulk Upload Validation**: Enhanced error reporting during Excel uploads (highlighting specific row errors).

## 📱 Low Priority
- **Mobile Companion App**: Simplified view for photographers to check schedules and venues on the go.
- **Client Portal**: Allow clients to log in and see their booking status and payment history.
